# Проект: Место

## Сдача проекта "Место". Описание.
 Проектная работа 5 включает в себя оформление сайта с применением адавтивной верстки и "оживлением" элементов.  Добавлены попап для создания новой карточки с фото и подписью и попап просмотра картинки. Первые шесть карточек заполняются автоматически на основе массива. Добавлена опция удаления карточки и лайк карточки. Все всплывающие окна получили функцию плавного открытия и закрытия. В работе были использованы: HTML, CSS. JavaScript 
 Цель проекта - отработать технологии JavaScript при взаимодействии с элементами.
